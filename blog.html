1. What is the DOM ? What is the purpose of DOM ?

Answer:

DOM is acronym of Document Object Model. When a web page losded, the browser create a tree model of the page it's called DOM model.
The purpose of DOM Model:
1. We can change or add or remove any element, style, text, attributes etc.
2. We can make a dynamic web page through DOM.
3. To make programmatically web page we use DOM.
4. It's help to load web page faster.

3. What is event Bubbling?

Answer:

When an event is triggered on an element, then it also triggers on all its parents elements is called event Bubbling.

4. What is Callback Function? what's its work?

Answer:

We can pass a function as arguments. Which function pass as a arguments of another function is called callback function.

Its work:
Callback function can be waited the result of previous function call and then excuate another function.
